# compositeSourceOver: Standard Alpha Overlay

### Foreground-Background Transparency Blending

The source over composite operation performs standard alpha blending where the source layer overlays the destination with alpha-based transparency mixing.

$$
\text{sourceOver} = src \times \alpha_{src} + dst \times \alpha_{dst} \times (1 - \alpha_{src})
$$
