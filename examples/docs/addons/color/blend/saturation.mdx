# blendSaturation: Chroma Channel Filter

### HSV Saturation Isolation System

Saturation blending isolates and replaces chroma intensity while preserving color identity and brightness through HSV channel manipulation. Base provides hue and value; blend contributes saturation data exclusively.

$$
HSV_{result} = (H_{base}, S_{blend}, V_{base})
$$

| Component      | Source | Visual Effect                 |
| -------------- | ------ | ----------------------------- |
| **Hue**        | Base   | Color identity preserved      |
| **Saturation** | Blend  | Chroma intensity replaced     |
| **Value**      | Base   | Brightness pattern maintained |

const fragment = () => {
        const brightPattern = float(0.4).step(uv.x.mul(15).sin().abs()).mul(0.6).add(0.4)

        const color = blendSaturation(
                hsv2rgb(vec3(0.6, 0.3, brightPattern)),

        )

}`}
