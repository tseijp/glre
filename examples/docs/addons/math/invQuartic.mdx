# invQuartic: Inverse Quartic Easing Function

### Double Square Root Transformation

The invQuartic function creates a smooth easing curve using nested square roots: $f(v) = \sqrt{1-\sqrt{1-v}}$. This produces a unique acceleration profile that starts slowly and accelerates quickly toward the end.

$$
f(v) = \sqrt{1-\sqrt{1-v}}
$$

Where the transformation domain spans $v \in [0,1]$ and produces a distinctive S-shaped curve with quantum-like transition characteristics.
