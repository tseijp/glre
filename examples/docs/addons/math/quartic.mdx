# quartic: Fourth-Order Polynomial Smoothstep with Quadratic Acceleration Dynamics

### Fourth-Order Smooth Interpolation

The quartic function provides smooth transitions using a fourth-order polynomial: $f(v) = v^2(2-v^2)$. This creates a different easing curve compared to cubic or quintic functions, with unique acceleration characteristics that can be useful for specific animation or transition effects.

Unlike conventional smoothstep functions, quartic introduces asymmetric acceleration profiles through its derivative $f'(v) = 4v(1-v^2)$, which reaches maximum velocity at $v = 1/\sqrt{3} \approx 0.577$. This mathematical property enables the creation of naturally occurring acceleration patterns that mirror physical phenomena in fluid dynamics and wave propagation systems.

The function's geometric characteristics include:

$$f(v) = v^2(2-v^2) = 2v^2 - v^4$$

$$f'(v) = 4v - 4v^3 = 4v(1-v^2)$$

$$f''(v) = 4 - 12v^2 = 4(1-3v^2)$$

The second derivative reveals inflection points at $v = \pm 1/\sqrt{3}$, creating natural transition zones that produce organic-feeling animation curves and spatial transformations.
