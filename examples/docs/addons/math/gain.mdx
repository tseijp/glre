# Dynamic Contrast Amplifier

### Intelligent Intensity Modulation

The gain function provides sophisticated contrast control that goes beyond simple linear adjustments. By applying asymmetric transformations to values above and below the midpoint, it creates dramatic visual intensity shifts that enhance both subtle details and bold contrasts simultaneously.

This mathematical approach enables precise control over visual punch, creating effects that range from subtle artistic enhancement to extreme psychedelic transformations, all while maintaining mathematical elegance and computational simplicity.

<FragmentEditor

const fragment = () => {
        const intensity = sin(iTime).add(1).div(2).mul(4).add(1)
        const pattern = sin(uv.x.mul(PI.mul(8))).mul(sin(uv.y.mul(PI.mul(6))))
        const enhanced = gain(pattern.add(1).div(2), intensity)
        return vec4(vec3(enhanced), 1)
}`}

Gain control applied independently to color channels creates spectacular chromatic explosions. Each color component responds differently to the gain curve, producing vibrant separations and unexpected color combinations that shift dynamically through the intensity spectrum.
