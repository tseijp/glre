# cubicInOut: Symmetric Sharp Transform Function

### Mathematical Foundation for Dual-Phase Sharp Motion

The `cubicInOut` function combines cubic acceleration and deceleration through piecewise third-degree polynomial construction. This creates symmetric motion profiles with dramatic transitions between phases.

$$
f(t) = \begin{cases}
4t^3 & \text{if } t < 0.5 \\
\frac{1}{2}(2t-2)^3 + 1 & \text{if } t \geq 0.5
\end{cases}
$$

The piecewise definition ensures continuous motion across the midpoint transition while maintaining cubic characteristics that create dramatic acceleration and deceleration phases.

### Velocity Profile Analysis

The derivative reveals symmetric quadratic acceleration and deceleration patterns around the midpoint with sharp transitions.

$$
\frac{df}{dt} = \begin{cases}
12t^2 & \text{if } t < 0.5 \\
3(2t-2)^2 & \text{if } t \geq 0.5
\end{cases}
$$

This creates maximum velocity at $t = 0.5$ with sharp velocity profiles that remain minimal near endpoints while building dramatic momentum at the center.

### Symmetry Properties with Sharp Transitions

The function exhibits perfect symmetry around the center point $(0.5, 0.5)$ while maintaining cubic curve characteristics that create dramatic visual impact.
