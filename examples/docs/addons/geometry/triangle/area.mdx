# area: Triangle Area Calculation

### Cross Product Method for Precise Triangle Area

The triangle area function implements the standard cross product formula for computing triangle area in 3D space. Given three vertices $A$, $B$, and $C$, the area is calculated as:

$$\text{Area} = \frac{1}{2} \|\vec{AB} \times \vec{AC}\|$$

where $\vec{AB} = B - A$ and $\vec{AC} = C - A$ represent the edge vectors from vertex $A$ to vertices $B$ and $C$ respectively.

The cross product $\vec{AB} \times \vec{AC}$ produces a vector perpendicular to the triangle plane whose magnitude equals twice the triangle area. Dividing by 2 yields the actual area.

const fragment = () => {

        const triangleArea = area(v1, v2, v3)
        const bary = barycentricFromPosition(v1, v2, v3, vec3(p, 0))
        const inside = bary.x.greaterThan(0).and(bary.y.greaterThan(0)).and(bary.z.greaterThan(0))
        const areaScale = triangleArea.mul(2).saturate()
        const insideColor = vec3(areaScale, areaScale.mul(0.8), areaScale.mul(0.5))
        const outsideColor = vec3(0.1, 0.2, 0.4)
        const triangleColor = outsideColor.select(insideColor, inside)
        const edges = min(min(
                length(p.sub(v1.xy)),
                length(p.sub(v2.xy))
        ), length(p.sub(v3.xy)))
        const vertices = smoothstep(0.05, 0.08, edges).oneMinus()
        const areaText = floor(triangleArea.mul(20)).div(20)
        const grid = sin(p.x.mul(20)).mul(sin(p.y.mul(20))).mul(0.1).add(0.9)
        return vec4(triangleColor.add(vertices.mul(0.7)).mul(grid), 1)
}`}
