# Installation

```ruby
npm i glre
```

or

```ruby
yaan add glre
```

## Install from CDN or static hosting

```html
<script type="module">
        import { gl } from 'https://cdn.skypack.dev/glre@latest'
        gl.setConfig({ id: 'myCanvas' })
</script>
```
